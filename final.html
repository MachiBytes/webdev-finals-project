My Activities

For the final activity 1, I had to install Laravel and Laragon. Afterwards, I created and opened a simple Laravel default project to make sure that my installation was working. It was my first time using Laragon and it definitely made it so much easier to start with Laravel, because back then, I had to manually modify the php.ini file to enable a lot of things for Laravel.

Download Activity 1

For the final activity 2, I had to create a simple Laravel project that has a login and registration system. This was a really fun and helpful activity because it takes so much off your plate as Laravel UI takes care of the authentication process for you. Not only does it provide controllers for authentication but it comes with views as well that are ready for you to customize. I will definitely use Laravel UI in my future projects.

Download Activity 2

This final activity 3 was a bit more challenging because I had to create a Laravel project with full CRUD functionality. It was another fulfilling activity and it helped me practice my skills with Laravel by making me connect my views to my controllers and models, which is something I haven't done much of before since I mainly use Laravel for REST API development.

This is the most challenging activity for the entire semester and it earns its right to be called a final project. There's just so much that went into this project and I'm glad that we were able to finish it. Splitting off the responsibilities between the group members was a bit challenging as we were often waiting on each other but it paid off because everyone was able to focus on just one aspect of the project and that allowed us to make the most of it. I'm also glad that we were able to finish it on time and I'm proud of the work that we put into it.
